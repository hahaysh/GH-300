프로젝트 구조 설명

최상위 폴더

  • README.md - 프로젝트 전반에 대한 설명 파일
  • ConsoleApp/ - 콘솔 애플리케이션(C# .NET 8.0)
  • WebApi/ - ASP.NET Core Web API 애플리케이션

ConsoleApp/

  • ConsoleApp.csproj - 콘솔 앱 프로젝트 파일
  • Program.cs - 콘솔 앱의 진입점(Main 함수)
  • bin/ - 빌드된 실행 파일 및 라이브러리(.dll, .exe 등) 저장 폴더
  • obj/ - 빌드 중간 산출물 저장 폴더

WebApi/

  • WebApi.csproj - Web API 프로젝트 파일
  • Program.cs - Web API의 진입점(Main 함수 및 Startup 구성)
  • appsettings.json, appsettings.Development.json - 환경별 설정 파일
  • WebApi.http - API 테스트용 HTTP 요청 샘플 파일
  • bin/ - 빌드된 실행 파일 및 라이브러리 저장 폴더
  • obj/ - 빌드 중간 산출물 저장 폴더
  • Properties/launchSettings.json - 개발 환경 실행 설정

빌드 및 실행

  • 각 프로젝트 폴더(ConsoleApp, WebApi)에서 dotnet build 또는 dotnet run 명령어로 빌드/실행

기타

  • 각 bin/obj 폴더는 빌드 시 자동 생성되며, 소스 관리에는 포함하지 않는 것이 일반적입니다.
